금촌 와와코칭센터
개념 간의 인과 관계를 명확히 이해하기 위해 각 주제를 도식화하는 습관은 복잡한 내용을 단순화하는 데 매우 효과적이다. 또한 정답을 외우는 데 급급하지 않고, ‘왜 이 답이 나왔는가’를 질문으로 반복하게 하는데, 모든 문장을 의문형 구조로 바꿔보는 연습을 시도한다. 학습 과정에서 감성의 조절은 지적 수행만큼 중요하다. 금촌 와와코칭센터은 학습 활동을 하나의 프로젝트로 간주하고 ‘Planning a Project’ 방식으로 단계를 분리하여, 목표 설정 → 자료 수집 → 실행 → 피드백 → 최종 점검이라는 흐름으로 진행함으로써 체계적인 사고력을 기른다. 금촌 와와코칭센터은 이렇게 탄력적인 설계는 매일 다른 상황 속에서도 학습의 일관성을 유지하는 비결이다. 또한 학습 중 느낀 점을 바로바로 메모로 기록해 두면, 나중에 정리할 때 빠르게 복원할 수 있으며, 생각이 떠올랐을 때 기록하는 습관은 창의적 사고의 흐름을 놓치지 않게 돕는다. 예를 들어, 시험 기간을 앞두고 학습자의 목표와 실력 수준에 맞춘 요약 자료를 제작하여 시간을 효율적으로 활용할 수 있도록 지원하였으며, 이 자료는 단순 요약을 넘어서 표와 그래프를 활용한 시각적 비교 분석까지 포함하였다.
